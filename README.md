TaksFlow (aplikasi daily taks reminder)
TaskFlow adalah aplikasi pengingat tugas harian yang dirancang untuk membantu pengguna mengatur, memantau, dan menyelesaikan pekerjaan dengan lebih terstruktur dan efisien. Aplikasi ini menyediakan ringkasan progres harian, daftar prioritas utama, serta fitur manajemen tugas berdasarkan tingkat urgensi (high, medium, dll). Dengan tampilan yang sederhana dan intuitif, TaskFlow memudahkan pengguna dalam menambahkan tugas baru, melihat status penyelesaian, serta menjaga produktivitas setiap hari.
